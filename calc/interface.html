# Интерфейс

# Основной экран

Основное окно калькулятора состоит из трех частей

# Общие параметры

Не имеют прямого влияния на товары. Задают … (необходимо уточнение)

# Параметры товаров

Задают базовые значения параметров товаров, такие как соотношение коэффициентов по продажам, транспортные затраты и прочее. Влияют на значения соответствующих параметров для каждого товара (подробнее в разделе Синхронизация).

Режим спроса определяет, какие данные будут использованы для рассчёта. В режиме Абсолютный используются данные … (необходимо уточнение), в режиме Исторический – исторические данные, полученные из Метрики.

# Товары

Представляют из себя список выбранных товаров, которые участвуют в рассчёте.

Каждый товар содержит блок Параметров и Результаты.

Предлагаемое количество товаров для закупки можно увидеть в правой верхней части таблицы с Результатами.

# Главное меню

Для перехода в меню необходимо нажать на кнопку menu в левой верхней части экрана

# Начало работы

Вначале необходимо выбрать товары, которые будут использованы для рассчёта. Это можно сделать через Каталог, который доступен либо через кнопку Открыть каталог, либо из Главного меню.

# Каталог

Представляет из себя древовидный список товаров.

Выбор необходимых товаров осуществляется с помощью клика либо по чекбоксу рядом с товаром, либо по чекбоксу рядом с категорией

Необходимые товары можно отфильтровать через поиск

# Синхронизация

Некоторые значения добавляемых товаров по умолчанию синхронизированы с Параметрами товаров. Такие поля отмечены галочкой check

Такимм образом, при изменении параметров через Параметры товаров, значения всех товаров, которые синхронизированы с общими параметрами, изменяются

Чтобы отвязать какое-либо поле конкретного товара от общих значений, достаточно ввести новое значение в это поле. При этом иконка check заменяется на sync

Для того чтобы восстановить привязку к значению из Параметров товаров, достаточно нажать на иконку sync рядом с выбранным полем

# Сортировка

Для удобства работы с выбранными товарами доступна сортировка по спросу или по цене. Соответствующие иконки расположены в правой верхней части раздела Товары.

При добавлении новых товаров они продолжают подчиняться выбранному режиму сортировки.

Сортировка по спросу зависит от режима, выбранного в Параметрах товаров. В “Историческом” режиме сортировка осуществляется относительно значения исторического спроса. В режиме “Абсолютный” – относительно абсолютного спроса.
